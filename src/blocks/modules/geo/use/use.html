How to use Ifplay_arrowSo in a few minutes?
Did you choose Ifplay_arrowSo in the WordPress admin panel? You can start producing geolocation-based content in 5 simple steps:
Step One
Create a new trigger, click on "Select a condition" and select "Geolocation". Then choose a continent, country or city
Step Two
In the Dynamic Content field - define the content that will be displayed if the condition is met. Then set the content in the default field.
Step Three
Click "Publish" and copy the shortcode created by the system to the desired location on the site.
