Common Questions

Is programming knowledge required?
Definately not! All it takes is to create the desired condition and define the content that will appear when it is realized.
How many placements can be added to each condition?
As many as you want!
What geolocation-based conditions can be set?
View geolocation-based content based on the user's continent, country, city, and time zone.
What content can be adapted to the user's geographical location?
All site content! Headlines, text, images, buttons, product and service pages and more.
Does the Ifplay_arrowSo plugin work with all WordPress themes?
Yes, so far we have not encountered a theme that the plugin does not work with. If for any reason the plugin does not work with your theme, our team will work with you to ensure that it happens.
How accurate is this?
Currently, there are no location-based services that offer a 100% accuracy level.
The Ifplay_arrowSo solution provides an extremely high level of accuracy, based on a variety of parameters.
